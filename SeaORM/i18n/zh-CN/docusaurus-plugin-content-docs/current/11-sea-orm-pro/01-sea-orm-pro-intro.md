# 🖥️ SeaORM Pro

![](https://www.sea-ql.org/blog/img/sea-orm-pro-light.png)

SeaORM Pro 是一个管理面板解决方案，让你能够快速轻松地为应用启动管理面板——无需前端开发技能（当然有的话更好）。

管理面板对于运营后端应用至关重要。但它往往被事后考虑，或者没有投入专门资源（即开发时间）来开发它们。

前端工程师通常专注于开发面向消费者的应用，这往往导致面向内部应用的 GUI 投入不足。

SeaORM Pro 旨在弥合这一差距，提供既快速实现又可靠长期使用的解决方案。

在[此处 (英文)](https://www.sea-ql.org/sea-orm-pro/docs/introduction/sea-orm-pro/)了解更多关于 SeaORM Pro 的信息。
