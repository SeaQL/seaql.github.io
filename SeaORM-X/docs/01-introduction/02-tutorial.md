# Tutorial & Examples

If you prefer a step-by-step tutorial on how to use SeaORM, check out the official [SeaORM Tutorial](https://www.sea-ql.org/sea-orm-tutorial/). There are also good tutorials written by [the community](https://github.com/SeaQL/sea-orm/blob/master/COMMUNITY.md#learning-resources).

The [SeaORM Cookbook](https://www.sea-ql.org/sea-orm-cookbook/) covers frequently asked questions and recommended practices.

For a minimal working example in a single file, try the [Quickstart](https://github.com/SeaQL/sea-orm-x/blob/master/examples/quickstart/src/main.rs).
